# gojo

## Project setup
### step1
```
yarn install
```

### step2
- download "furuyoni commons"
  - https://main-bakafire.ssl-lolipop.jp/furuyoni/na/rule.html
- copy required card images
  - `furuyoni_commons_na/furuyoni_na/cards/*.png` to `src/assets/cards/`
